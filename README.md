## Проекты

### [Проект 1. Анализ вирусного генома (FASTA)](fasta-pb2-gene-analysis/)

**Задача:** Проанализировать ген вируса гриппа A (сегмент PB2) и найти потенциальную терапевтическую мишень.

**Результаты:**
- Расчёт GC-состава — **42.50%** (в пределах нормы)
- Поиск открытых рамок считывания (ORF) — найдено **5 ORF**
- Выделен самый длинный ORF — **2280 нт** (кодирует белок PB2, 760 аминокислот)
- Потенциальный ген-мишень сохранён в `potential_target.fasta`

**Инструменты:** Python, matplotlib, numpy

---

### [Проект 2. Анализ качества FASTQ и фильтрация ридов](fastq-quality-control-pipeline/)

**Задача:** Контроль качества данных секвенирования (RNA-seq) перед выравниванием на референсный геном.

**Результаты:**
- Парсинг FASTQ-файлов (поддержка .gz)
- Расчёт Phred quality scores
- Фильтрация по качеству и длине (Q > 25, длина > 50 нт, >85% хороших позиций)
- Визуализация качества по позициям (перцентили, пороги Q20/Q25/Q30)
- Сохранено **75%** ридов после фильтрации

**Инструменты:** Python, matplotlib, numpy
