# Поиск потенциальных терапевтических мишеней в геноме вируса гриппа A (сегмент полимеразы PB2)

## Актуальность

Грипп — это острое вирусное заболевание, против которого не существует универсальной вакцины из-за высокой изменчивости вируса. Белок **PB2** (субъединица РНК-зависимой РНК-полимеразы) отвечает за связывание кэп-структуры клеточных мРНК и является ключевым фактором вирулентности, особенно для штаммов, способных заражать млекопитающих.

## Цель исследования

Проанализировать нуклеотидную последовательность сегмента 1 (ген PB2) вируса гриппа A, чтобы:

1. Оценить стабильность генома (через GC-состав)
2. Найти потенциальные открытые рамки считывания (ORF), кодирующие белки
3. Выделить наиболее перспективный фрагмент ДНК как кандидата для дизайна противовирусных молекул

## Гипотеза

Участки с GC-составом около 45–50% являются оптимальными для дизайна siRNA. В геноме вируса должны присутствовать длинные открытые рамки считывания (ORF), кодирующие функциональные белки.

## Данные

| Параметр | Значение |
|----------|----------|
| **Источник** | NCBI Nucleotide Database |
| **Идентификатор** | NC_007373.1 |
| **Организм** | Influenza A virus (A/New York/392/2004, H3N2) |
| **Сегмент** | 1 (PB2 — РНК-полимераза) |
| **Размер** | 2341 нуклеотид |
| **Формат** | FASTA |

## Методы

- Расчёт GC-состава (стабильность ДНК/РНК)
- Визуализация GC-распределения по геному (скользящее окно)
- Поиск открытых рамок считывания (ORF) на прямой и обратной цепях
- Трансляция ДНК в аминокислотную последовательность

## Результаты

| Показатель | Значение |
|------------|----------|
| Длина генома | 2341 нуклеотид |
| GC-состав | 42.50% |
| Найдено ORF (длиннее 150 нт) | 5 |
| Самый длинный ORF | 2280 нт (760 аминокислот) |

## Визуализация

Программа создаёт график распределения GC-состава по геному (`results/gc_content_plot.png`), который показывает локальные участки с высоким/низким GC-содержанием.

## Интерпретация результатов

### GC-состав 42.50%
- Нормальный GC-состав (40-60%) — оптимален для большинства экспериментов
- Ген PB2 подходит для дизайна антисмысловых олигонуклеотидов

### Поиск ORF
- **Первый ORF (2280 нт → 760 а.к.)** — полноразмерный белок PB2 (РНК-полимераза). Это **главная терапевтическая мишень**
- Остальные ORF короче 300 нт — вероятно, нефункциональные фрагменты

## Выводы

1. GC-состав гена PB2 составляет **42.50%** — в пределах нормы, подходит для дизайна siRNA
2. Обнаружена полная ORF длиной **2280 нуклеотидов** — соответствует белку PB2 длиной 760 аминокислот
3. Выделенный фрагмент (`data/potential_target.fasta`) — кандидат для разработки противовирусных молекул
4. Белок-мишень сохранён в `data/translated_protein.fasta`
